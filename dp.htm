Microsoft DP 系列認證

DP 系列是 Microsoft Data Platform（資料平台）相關認證， 主要與資料庫、資料工程與資料處理技術有關。

Data Platform 技術

  • SQL Server
  • Azure SQL Database
  • Azure Data Factory
  • Azure Synapse Analytics

常見 DP 認證

  • DP-900：Azure Data Fundamentals
  • DP-203：Data Engineering on Microsoft Azure

考試方式

  • 線上或考場考試
  • 選擇題、案例題
  • 部分考試包含實務情境題
  • 滿分 1000 分，700 分及格

適合的人

  • 資料工程師
  • 資料庫管理員
  • 後端工程師
  • 想學習雲端資料平台的人
